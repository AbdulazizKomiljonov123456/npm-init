• Ссылка Описание Установка через терминал
  • npm Установка пакетного менеджера npm в вашу папку (директорию) npm init
  • gulp-cli - Установка gulp-cli глобально, на все проекты npm install gulp-cli -g
  • gulp - Установка gulp npm install gulp -D
  • gulpfile.js - Установка файла для работы с gulp npx -p touch nodetouch gulpfile.js
  • gulp-autoprefixer - Добавляет приписки к свойсвам в CSS для поддержки их во всех браузерах npm install --save-dev gulp-autoprefixer
  • @babel/core - Ядро плагина babel для работы всех его доп.модулей npm install --save-dev @babel/core
  • gulp-babel - Нужен для работы babel в task-менеджере gulp npm install --save-dev gulp-babel
  • @babel/preset-env - Создает из новой версии JS более страрую для работы в старых браузерах npm install --save-dev @babel/preset-env
  • babel-preset-minify - Создает минифицированную версию JavaScript npm install babel-preset-minify --save-dev
  • gulp-group-css-media-queries - Обрабатывает все медиа запросы и помещает их в конец CSS для правильной работы npm install --save-dev gulp-group-css-media-queries
  • browser-sync- Проверяет файлы на изменения и делает запуск сервера npm install browser-sync gulp --save-dev
  • gulp-sourcemaps- Даёт возможность просмотреть файл SCSS в браузере npm i gulp-sourcemaps --save-dev
  • gulp-sass- Нужен для работы с технологией SCSS npm install node-sass gulp-sass --save-dev
  • gulp-include- Нужен для работы с подключениями файлов npm install gulp-include
  • gulp-load-plugins- Запускает все плагины GULP npm install --save-dev gulp-load-plugins
  • gulp-rename- Переименовывает файлы npm install gulp-rename
